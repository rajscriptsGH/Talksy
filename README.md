# Talksy

Talksy is a real-time chat and video app for fast and simple communication.

## Features

- 💬 Instant messaging  
- 📹 Video calls  
- ⚡ Real-time updates  
- 🎨 Clean and modern UI  

## Tech Stack

- React, JavaScript
- Node.js, Express  
- WebSocket / WebRTC  
- MongoDB

## Getting Started

1. Clone the repository  

   ```bash
   git clone https://github.com/your-username/talksy.git
   ```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
